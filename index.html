基底関数による非線形性の有限要素近似

§基底関数

まず、長さ\(d\)の領域を\(m\)個の有限要素に等分割し(\(\Delta d=d/m\))、 \(m+1\)個の有限要素のつなぎ目(節)の位置を\(x=p_i\)として、 節\(p_i\)における基底関数\(\phi_i(x)~~(i=0\cdots m)\)を次式で定義します。

\(i=0\)

\begin{align} \left. \begin{array}{@{\,}rl} \phi_{0}(x)&=~1&~(x \le p_0),\\ \phi_{0}(x)&=~\frac{p_1-x}{p_1-p_0}&~(p_0 < x \le p_1),\\ \phi_{0}(x)&=~0&~(p_1 < x). \end{array} \right\} \label{eq:BaseFunc_1} \end{align}

\(i=1,\cdots,m-1\)

\begin{align} \left. \begin{array}{@{\,}rll} \phi_{i}(x)&=~\frac{x-p_{i-1}}{p_i-p_{i-1}}&~(p_{i-1} < x \le p_i),\\ \phi_{i}(x)&=~\frac{p_{i+1}-x}{p_{i+1}-p_i}&~(p_i < x \le p_{i+1}),\\ \phi_{i}(x)&=~0&~(x \le p_{i-1},~p_{i+1} < x). \end{array} \right\} \label{eq:BaseFunc} \end{align}

\(i=m\)

\begin{align} \left. \begin{array}{@{\,}rl} \phi_{m}(x)&=~0&~(x \le p_{m-1}),\\ \phi_{m}(x)&=~\frac{x-p_{m-1}}{p_m-p_{m-1}}&~(p_{m-1} < x \le p_m),\\ \phi_{m}(x)&=~1&~(p_m < x). \end{array} \right\} \label{eq:BaseFunc_m} \end{align}

基底関数\eqref{eq:BaseFunc_1}～\eqref{eq:BaseFunc_m}をグラフで表現すると次の図になります。

§基底関数による関数の有限要素近似

基底関数\(\phi_i(x)\)を用いて関数\(f(x)\)を次式の\(\hat f(x)\)で有限要素近似できます。

\begin{align} \hat f(x) = \sum_{i=1}^n w_i\phi_i(x) \label{eq:ApproXFunc} \end{align}

変位\(x \in [p_{i-1},~p_{i}]\)のとき、関係する基底関数は\(\phi_{i-1}(x)\)と\(\phi_i(x)\)だけになり(他は\(0\))、 関数\(f(x)\)有限要素近似\eqref{eq:ApproXFunc}は次式となります。

\begin{align} \hat f(x) = \hat w_i\phi_{i-1}(x) + \hat w_i\phi_i(x)~~~(i=1,\cdots,m) \label{eq:Func_eta2} \end{align}

\(\phi_i(p_i)=1\)なので、基底関数\(\phi_i(x)\)の荷重\(w_i=f(p_i)\)になります。
